Getting Started
===============

Prerequisites
-------------

This library require:

- jquery
- momentjs
- hammerjs
- jquery-knob

Installation
------------

Installation is a quick:

Add library in your `bower.json`:

```json
{
    "dependencies": {
        "sonatra-jquery-datetime-picker": "latest"
    }
}
```

Launch bower:

```bash
$ bower install
```

Dev installation
----------------

### Use NPM

```
npm install
```

### Use Bower

```
bower install
```

### Use grunt

```
grunt serve
```
